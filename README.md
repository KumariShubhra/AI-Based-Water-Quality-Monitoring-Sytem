# 💧 AI-Based Water Quality Prediction System

## 📌 Overview

This project is a Machine Learning-based web application that predicts water quality using key environmental parameters. It is built using Flask and Random Forest algorithm, providing real-time predictions along with risk analysis and visualization.

---

## 🚀 Features

* ✅ Water Quality Prediction using ML model
* 📊 Graphical analysis of parameters
* ⚠️ Risk level indication (Safe / Unsafe)
* 📈 Feature importance visualization
* 📄 PDF report generation
* 💡 User-friendly dashboard interface

---

## 🛠️ Technologies Used

* Python
* Flask
* Scikit-learn (Random Forest)
* Pandas & NumPy
* Matplotlib
* HTML, CSS

## ▶️ How to Run the Project

1. Clone the repository

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python app.py

4. Open browser:
   http://127.0.0.1:5000/
